## Amazon S3 (Simple Storage Service)

### 1. 소개

- 객체 스토리지 서비스 (파일 저장)
- 리전 별 저장, 최소 3개의 가용 영역에 분산 저장
- 무제한 용량, 단일 객체 최대 5TB 용량

### 2. 구조

- Bucket
  - S3 의 저장 공간을 구분하는 단위이며, 폴더와 같은 개념
  - 리전과 관계 없이 전 세계에서 고유한 값을 사용해야 함


- S3 객체 구성
  - Owner: 소유자
  - Key: 파일명
  - Value: 데이터
  - Version ID: 파일 버전
  - Metadata: 파일 정보
  - ACL: 파일 권한
  - Torrents: 토렌트 정보


- 보안 설정
  - 버킷 생성 시 Private 기본값
  - MFA 를 통한 삭제 방지
  - 액세스 로그 생성, 전송
  - 객체 단위, 버킷 단위 구성 가능
    - Bucket policy: 버킷 단위
    - ACL(Access Control List): 객체 단위


### 3. 정적 호스팅
- S3 에 정적 리소스를 업로드하여 호스팅
- 불특정 다수에 대한 권한 허용
  - S3 Block public access 해제
  - Bucket policy 정책 허용
