## AWS Lambda

### 1. 소개

- 애플리케이션, 백엔드 서비스에 대한 코드 실행이 가능한 서버리스 서비스
- 런타임 상에서 작은 단위의 함수 코드를 실행 가능 (최대 실행 시간 15분 이내)

### 2. 구조

- 사용 가능 런타임
  - Java
  - C#
  - Go
  - Node.js
  - Python
  - Ruby
  - 커스텀
- Ephemeral storage: 임시 스토리지
- Deployment package
  - 코드와 실행을 위한 파일을 묶은 패키지
  - 인라인: 별도 업로드 없이 람다 콘솔에서 코드 수정 (파일당 최대 3MB)
  - zip 파일: 코드와 실행을 위한 파일을 묶은 압축 파일 (최대 250MB)
  - S3 업로드
  - Container image: 직접 OS/람다 런타임 등의 파일을 담은 이미지로 구동 (최대 10GB)

### 3. AWS API Gateway

- AWS 서비스, 외부 서비스를 위한 API 를 생성/관리하기 위한 서비스
- HTTP/Websocket 프로토콜 지원
